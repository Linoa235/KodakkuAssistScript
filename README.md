# KodakkuAssistScript

------> This is simply a translation of the scripts made by the Koda community from Chinese to English (there may be a few translation errors… or not).

Each script listed in this repo has not been made by me, but by the people who are listed in the Kodakku community. I only translate the scripts to make them available as much as possible in other languages. So please thank the authors listed in the Kodakku community.


A repository for storing scripts written for the Dalamud plugin KodakkuAssist.
